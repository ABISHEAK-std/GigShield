┌──────────────────────────────────────────────────────────────────┐
│  ⚠️ TROUBLESHOOTING:                                            │
└──────────────────────────────────────────────────────────────────┘

Backend won't start?
  → Check Java: java -version (need 17+)
  → Check Maven: mvn -version (need 3.8+)
  → Try manual build:
    cd D:\ALLIXA\allixia-backend
    mvn clean install -DskipTests
    mvn spring-boot:run

Frontend won't load?
  → Check backend: http://localhost:8082/api/health
  → Open browser console (F12) for errors
  → Try different browser (Chrome, Edge, Firefox)
  → Clear browser cache

Database errors?
  → Database is pre-configured (Neon Cloud)
  → No local setup needed
  → Credentials in application.properties
  → Flyway auto-creates tables on first run
